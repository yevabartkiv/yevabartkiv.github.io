      • IdM Modules

        In this section we provide Marketing information about application Modules.

        • Module List

          Module list provides you short information about all IdM Modules. To get more information about specific Module click on its Title Name.

        • m01. Recertification

          Role recertification module approves assigned user roles again. When user has a lot of assigned roles for a long time, we want to check these assigned roles periodically (in a half year interval for security reasons), if some assigned role has to be already removed. Currently valid manual direct assigned roles are checked - only manual roles can be assigned and stay assigned, after user is changed some way (e.g. user contract is excluded, work position was changed).
