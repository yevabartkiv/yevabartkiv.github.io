IdM Features

In this section we provide Marketing information about application Features

Features are the important parts of IdM application and its standalone Modules. As mentioned in IdM Modules section, there's no need to purchase all Modules to run the application. To get solution tailored to your company's needs, look at Feature list section or use IdM Configurator (in developing phase). With the Configurator, you can select the enterprise tasks you want to accomplish using the IdM application. Based on your selection, you’ll receive the necessary IdM modules and the final application price.
